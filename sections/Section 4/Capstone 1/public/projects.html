Raspberry Pi 8-Node Cluster

Built an 8-node Raspberry Pi cluster for experimenting with distributed computing and parallel processing. Utilized MPI (Message Passing Interface) for inter-node communication. The cluster was used for tasks like rendering complex 3D models and running simulations.

Technical Details
  • Raspberry Pi 4 Model B
  • Raspbian OS
  • MPI (Message Passing Interface)
  • Docker

VMware ESXi Server on Dell PowerEdge

Set up a VMware ESXi virtualization server on a repurposed Dell PowerEdge tower server. This involved installing and configuring ESXi, creating virtual machines, and managing resources. The server was used for hosting personal projects and experimenting with different operating systems.

Technical Details
  • Dell PowerEdge T320
  • VMware ESXi 7.0
  • Virtual Machines: Windows Server, Ubuntu Server

Home Automation System

Developed a home automation system using a Raspberry Pi and various sensors. The system controls lighting, temperature, and security, and can be accessed remotely. It integrates with various smart home devices.

Technical Details
  • Raspberry Pi 4
  • Python
  • MQTT
  • Various sensors (temperature, light, motion)
